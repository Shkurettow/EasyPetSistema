# EasyPetSistema
Em Desenvolvimento - O EasyPet é um software desktop desenvolvido para otimizar a gestão de pet shops e clínicas veterinárias. O objetivo principal é oferecer uma interface intuitiva para que o atendente possa realizar cadastros de animais e agendamentos de serviços de forma rápida, substituindo processos manuais por um banco de dados digital seguro.

 Tecnologias Aplicadas
* Java (Swing)
* MySQL (Banco de Dados)
* JDBC (Conexão)
* NetBeans IDE

 Objetivo do Software
 
O sistema visa facilitar a gestão de uma pet shop, permitindo o controle de cadastros e agendamentos de forma digital e organizada.

 Funcionalidades (Requisitos)
* **RF002:** Cadastro de Pets (Nome, Espécie, Raça).
* **RF004:** Agendamento de Serviços.
* **RNF003:** Tratamento de erros em banco de dados.
